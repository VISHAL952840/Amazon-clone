# Amazon Clone (HTML & CSS)

This project is a static clone of the Amazon homepage built using only HTML and CSS.
The goal of this project was to practice frontend layout, styling, and responsive design.

## Technologies Used
- HTML5
- CSS3

## Features
- Amazon-like homepage UI
- Navigation bar
- Product sections
- Responsive layout (basic)

## What I Learned
- Page structuring with HTML
- Flexbox & CSS Grid
- Responsive design basics
